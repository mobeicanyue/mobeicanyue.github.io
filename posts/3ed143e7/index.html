博客的 SEO 优化

本文最后更新于 2024年1月19日 17:48

写博客的目的是为了记录自己的学习过程，分享日常。而想要让更多的人看到自己的博客，就需要做一些 SEO 优化，提升自己的博客在搜索引擎上的排名。
特别是如果你的博客是部署在 GitHub Pages 上的，那么你就更需要注意了。因为百度被 Github 反爬了（参考 Github 的 robots.txt），你的博客在百度上是不会被收录的。有一个自己的域名是最好的。

使用百度搜索确实无法搜索到自己部署在 Github Pages 的博客，但是使用谷歌和必应搜索是可以的。

1. 安装 hexo-generator-sitemap

然后 执行

就可以在 public 目录下看到 sitemap.xml 文件了。

2. 配置 robots.txt

在 source 目录下新建 robots.txt 文件，内容如下：

User-agent: * 表示允许所有的搜索引擎爬取你的网站
Disallow: 为空 表示不设置爬取限制
Sitemap: 后面的链接是你的 sitemap.xml 文件的链接，这样搜索引擎就可以通过 sitemap.xml 文件来爬取你的网站了。

3. 搜索引擎提交

3.1 百度

百度搜索资源平台：https://ziyuan.baidu.com/site/index

在提交网站之前需要填写一些个人信息

3.1.1 输入网站地址

3.1.2 勾选网站类型

3.1.3 验证网站

下载文件放到网站根目录下，部署之后，点击验证

注意：为保持验证通过的状态，成功验证后请不要删除 HTML 文件

如果你是使用的 GitHub Pages 部署的博客，那么不要忘记在构建完成之后，将下载的文件放到 public 目录下。

3.2 谷歌

谷歌搜索控制台：https://www.google.com/webmasters/tools/home?hl=zh-CN

3.3 必应

必应站长平台：https://www.bing.com/webmasters/home
