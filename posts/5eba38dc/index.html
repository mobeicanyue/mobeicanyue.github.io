1.2 安装方式二：解压 GitHub 源码包

下载 主题最新 release 版本 解压到 themes 目录，并将解压出的文件夹重命名为 fluid。注意，这种方式安装的主题自带配置文件 _config.yml，位于主题根目录，不需要手动创建。

这种方式适用于想灵活修改主题的用户，因为 Fluid 更新较慢，如果你有一些自定义需求，可以直接修改源码。
